# Картинная галерея в Тамбове

Готовый учебный сайт в формате HTML/CSS/JS о Тамбовской областной картинной галерее.

## Как открыть

Дважды щёлкните `ОТКРЫТЬ САЙТ.bat` или откройте `index.html` в браузере.

## Страницы

- `index.html` — Главная
- `about.html` — О галерее
- `media.html` — Фото, панорамы и видео
- `scenario.html` — Сценарий посещения
- `scenario-en.html` — Visit Scenario
- `interview.html` — Интервью
- `interactive.html` — Интерактив
- `maps-sources.html` — Карты, маршруты, QR-код и источники

## Что входит

- 22 фотографии из папки `photos`
- 2 панорамы из папки `photos`
- Rutube-видео
- страница интервью без вопросов и ответов, только видео и подпись
- русский и английский сценарии посещения
- eLIBRARY, Scratch, Tinkercad, пазл, ребусы и кроссворд
- Google Maps, Яндекс Карты, маршруты, QR-код и список литературы

Учебный проект, 2026.
